=== G Analyticst ===
Donate link: http://www.ivystar.jp/

G AnalyticstはGoogle Analyticsのサイト分析結果をWordPressの管理画面で確
認することができるWordPressのプラグインです。


== Description ==

インストールが完了したら管理画面の｢ダッシュボード｣から｢G Analytics｣を選
択し、｢設定｣のリンクをクリックしてください。

設定画面では事前に登録を行ったGoogle AnalyticsのE-Mailとパスワードを入
力し[変更を保存]を一度押してください。プロファイル欄に登録済みプロファ
イルがプルダウンで表示されますので一覧から選択して再度[変更を保存]ボタ
ンを押してください。

｢Google Analyticsのリンクパス｣はGoogle AnalyticsのURLをお好みで入力して
ください。

入力が完了したら｢変更を保存｣をクリックします。以上で設定は完了です。

｢設定｣以外では分析期間をプルダウンから選択し、リンクをクリックしてGoogle
Analyticsの内容を確認してください。

また、テンプレート用のアクセスカウンターも用意しました。設定画面に使用
方法を明記してあります。

== Installation ==

下記からダウンロードした後、回答して利用しているWordPressのプラグイン用
ディレクトリ(通常は/wp-content/plugins/)へディレクトリごとアップロード
してください。

アップロードが完了したら管理画面の｢プラグイン｣で｢G Analytics｣を使用する
ように設定してください。

これでインストールは完了です。
